# zh-official-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源形式存放，不绑定任何特定域名或网站，主要用于资料的整理与展示。

## 项目简介

这是一个通用型 HTML 页面归档仓库。仓库内包含一系列独立页面，每个页面均以 HTML 文件形式存在，可通过浏览器直接打开查看。项目不针对任何具体网站或服务，仅为便于组织、维护和分享静态内容而设立。

## 目录说明

```
zh-official-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（如图片、样式、脚本等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- `pages/`：主要页面存放目录，每个 HTML 文件代表一个独立页面。
- `assets/`：页面所需的辅助资源文件，按需组织子目录。
- `index.html`：可选入口页面，用于展示页面列表或导航。

## 页面归档说明

- 所有页面均为独立 HTML 文件，无后端依赖。
- 页面内容仅供归档参考，不保证实时更新或与任何外部来源一致。
- 页面之间的链接或引用仅限仓库内部资源，不指向外部网站。
- 归档内容可能随项目维护而调整，包括新增、修改或移除页面。

## 维护说明

- 本仓库由项目维护者不定期更新。
- 欢迎通过 Issues 提交问题或建议，但请注意不涉及具体域名或营销内容。
- 提交 Pull Request 时，请确保新增或修改的页面符合归档规范，不包含外部链接或商业推广信息。
- 仓库维护者保留对内容进行审核和调整的权利。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可将仓库部署到任何静态托管服务，以在线方式访问。
